Programação Orientada a Objetos com Dart

Uma classe em Dart é a planta de construção (ou molde) para criar objetos. É extremamente importante entender que tudo em Dart é um objeto.

1. Definindo uma Classe

Para criar uma classe, usamos a palavra-chave class.
Imagine a classe Carro. Dentro dela, definimos uma variável de programa chamada modelo para armazenar o nome do veículo.

Classes são fundamentais para organizar código complexo de forma legível.

2. Exemplo Prático

Veja o código abaixo:
class Carro { String? modelo; }

Se você digitar dart run main.dart no seu terminal, poderá ver a execução do seu script.
A saída esperada no console (o output) seria algo como:

Instância de 'Carro' criada com sucesso!

3. Notas de Rodapé e Detalhes

Nota: O Dart suporta null safety por padrão.
Lembre-se: A convenção para nomes de classes é UpperCamelCase.

As fórmulas de performance de memória podem envolver cálculos complexos como:
Memória total = Objetos n + Cache.
